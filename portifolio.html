Projeto

Calculadora de Linha de Comando em C
  • Desenvolvi uma aplicação executada via console para realizar operações matemáticas fundamentais. Meu papel envolveu desde a estruturação do algoritmo até a implementação do código final. Utilizei estruturas de repetição (como do-while) para criar um menu interativo e persistente. Um desafio técnico que superei foi a implementação de validação de dados de entrada, garantindo que o programa não travasse ao tentar realizar divisões por zero, utilizando condicionais (if/else) para tratar essas exceções.
  • Linguagem C, Compilador GCC, Biblioteca Standard Input/Output (stdio.h) e Lógica de Programação Estruturada.
Revitalização de Computadores de Armazém
  • Este projeto teve como objetivo recuperar e colocar novamente em funcionamento diversos computadores antigos que estavam armazenados há anos em um depósito. A proposta era avaliar o estado das máquinas, realizar manutenção e tentar reaproveitar os equipamentos sempre que possível. Durante o processo, participei da análise e montagem dos computadores, identificando problemas de hardware e substituindo componentes defeituosos. Muitos dos equipamentos apresentavam falhas nas fontes de alimentação, processadores, memórias e outros componentes essenciais, o que exigiu diagnóstico cuidadoso e testes constantes.
  • Montagem e manutenção de hardware, Diagnóstico de componentes (CPU, memória RAM, fonte de alimentação, placa-mãe), Sistemas operacionais Windows e Linux, Ferramentas de teste e manutenção de hardware e Metodologias de troubleshooting (identificação e resolução de problemas)
